### 枚举类型
```ts
enum ColorsEnum {
    white = '#ffffff',
    black = '#000000',
}

type Colors = keyof typeof ColorsEnum;
// like
type Colors = 'white' | 'black'

// typeof ColorsEnum =>
type ColorsEnum = {
  white: '#ffffff'
  black: '#000000'
}
// keyof typeof ColorsEnum => 
type Colors = 'white' | 'black'
```

::: tip
`enum` 在编译时被用作类型，用来实现常量的类型安全，但是它们在运行时被视为对象。
这是因为，当 Typescript 代码被编译为 Javascript 时，它们会被转换为普通对象。
:::
